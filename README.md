# E-Commerce Superstore Analysis

## Project Overview

This project analyzes retail sales data from a Superstore dataset using Python and Power BI.

## Tools Used

* Python
* Pandas
* Matplotlib
* SQLite
* Power BI

## Key Analysis

* Sales Analysis
* Profit Analysis
* Category-wise Performance
* Region-wise Performance
* Monthly Sales Trends
* Top Products Analysis

## Dashboard Features

* Total Sales KPI
* Total Profit KPI
* Total Orders KPI
* Category Filter
* Region Filter
* Segment Filter
* Interactive Charts

## Key Insights

* Technology generated the highest sales.
* Sales varied significantly across regions.
* Monthly sales trends revealed seasonal patterns.
* A small set of products contributed heavily to revenue.

## Project Structure

* Data Cleaning
* Exploratory Data Analysis
* Dashboard Development
* Business Insights
